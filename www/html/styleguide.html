Style Guide

A guide to the base markup and custom patterns/modules used throughout the site.

According to Atomic Design, we start with global Atoms. Atoms are our HTML elements, such as a form label, an input, a button or a table. Atoms can also include more abstract elements like color palettes, fonts and even more invisible aspects of an interface like animations.

1. Grid, colors and font stacks

1.1 Grid

6 columns by default (for demo only). Can easily be changed in /scss/_settings.scss. Then go to /scss/components/_layout.scss, to generate new styles. Finally, update markup in /html/patterns/grid.html.
